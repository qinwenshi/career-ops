## 核心设计原则

| 层 | 关键决策 |
|----|---------|
| **提取** | Playwright 优先处理 SPA；三级降级保证覆盖率；本地 JD 存 `jds/` 目录 |
| **匹配** | `_shared.md` 定义原型 → `oferta.md` 驱动 A-F 六块评估 |
| **数据** | 指标不硬编码，每次从 `cv.md` + `article-digest.md` 实时读取 |
| **写入保护** | `applications.md` 只通过 TSV → `merge-tracker.mjs` 写入，防并发污染 |
| **验证** | `node verify-pipeline.mjs` 检查状态、重复项、报告链接完整性 |
